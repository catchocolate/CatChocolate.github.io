# 作品介绍
本作品为昆士兰大学工程学院短期项目，主要内容为设计并制作交互性为主的网站。本人与其他三人构成小组，在小组中的主要职责为概念设计、UI、UX以及部分前端编程。以下为该作品的介绍。

## 项目主题
本网站交互主题为：探索地图、解谜的游戏网页。游戏流程为下：

* 坐时光机穿越回战争年代试图找回图书馆丢失的二战士兵照片
* 探索现有地图中的不同区域
* 进入房间，发生对话/探索房间，寻找沙漏的线索
* 利用沙漏来点亮士兵照片，查看士兵故事
